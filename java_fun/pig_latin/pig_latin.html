According to the Perdue University website, Pig Latin has three rules, when translating from English.

  1. If the first letter is a consonant followed by a vowel: take the first letter and put it at the end of the word. Then add 'ay' to the end.
  2. If the first two letters are consonants: take the first two letters and put them at the end of the word. Then add 'ay' to the end.
  3. If the first letter is a vowel: add 'way' to the end.
